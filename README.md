# Real-Time Sales Analytics Pipeline

## 🚀 Overview
Designed and implemented an end-to-end data analytics pipeline to transform raw sales data into actionable business insights through automated ETL processes and interactive dashboards.

## 🧩 Problem Statement
Organizations often struggle with delayed and inconsistent reporting due to fragmented data sources and manual processes, limiting their ability to make timely, data-driven decisions.

## ⚙️ Solution
Developed a scalable data pipeline that:
- Extracts and processes raw sales data using Python
- Transforms and loads data into structured formats using SQL
- Implements a star schema data model for efficient querying
- Visualizes key business metrics through an interactive Power BI dashboard

## 📊 Key Features
- Automated ETL pipeline for continuous data processing  
- Advanced SQL queries for KPI computation (revenue, growth, customer trends)  
- Data modeling using fact and dimension tables  
- Interactive dashboard for real-time performance monitoring  

## 📈 Key Insights
- Identified top-performing regions contributing significantly to total revenue  
- Highlighted seasonal sales trends impacting business performance  
- Detected underperforming product categories for optimization opportunities  

## 💼 Business Impact
- Reduced manual reporting effort and improved efficiency  
- Enabled faster decision-making through real-time insights  
- Improved visibility into sales performance across multiple dimensions  

## 🛠 Tech Stack
- SQL (PostgreSQL / MySQL)
- Python (Pandas, NumPy)
- Power BI
- Data Modeling (Star Schema)
- ETL Pipeline Development

## 📁 Project Structure
- `data/` → Raw and processed datasets  
- `python/` → Data cleaning and ETL scripts  
- `sql/` → Analytical queries  
- `dashboard/` → Power BI files
